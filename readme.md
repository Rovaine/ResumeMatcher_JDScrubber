## Installation Steps

#### 1. Create the virtual environment
on windows:
```
python -m venv env

```
on linux:
```
python3 -m venv venv
```
#### 2. Activate the virtual environment
on windows:
```
env\Scripts\activate

```

on linux:
```
source venv/bin/activate

```

#### 3. Install the requirements
```
pip install -r requirements.txt

```
